Autres procédures de login

Mobile ID

Le login via téléphone mobile et Mobile ID fonctionne avec tous les téléphones mobiles dont la carte SIM est compatible avec Mobile ID.

Lecteur de PostFinance Card

Le login via le lecteur de PostFinance Card jaune et la PostFinance Card est adapté à tous ceux qui n’ont pas de téléphone mobile.
